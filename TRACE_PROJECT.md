# TRACE — Project Tracker
**Last Updated:** June 2026  
**Repo:** https://github.com/RevanthBoina/TRACE  
**Purpose:** Anti-piracy platform — creators upload videos, get invisibly watermarked, register platform links, system monitors 24/7 for unauthorized copies.

## VideoSeal Model
- File: `backend/ckpts/y_256b_img.jit` (218MB)
- Downloaded from: `https://dl.fbaipublicfiles.com/videoseal/y_256b_img.jit`
- **NOT in git** (gitignored) — downloaded by `setup.sh` every fresh session
- Uses TorchScript — no `pip install videoseal` needed
- Runs on CPU (ARM compatible)
- 256-bit watermark payload per video

---

## Fresh Session Setup
Every new session run this from project root:
```bash
chmod +x setup.sh && ./setup.sh
```
This does:
1. Pulls all secrets from AWS Parameter Store → writes `.env`
2. Downloads VideoSeal model if missing

**Requirement:** AWS CLI must be configured (`aws configure`) with valid credentials.

## Key Rules
- **Never store video files** — only fingerprints and metadata in DB
- **Free tier watermark:** 1 in 10 frames (test with all frames first, optimize later)
- **Paid tier watermark:** 5 in 10 frames
- **Detection logic:** 2 of 3 signals must match (VideoSeal + DINOHash + audio)
- **Scan frequency:** YouTube every 6hrs, Instagram every 12hrs
